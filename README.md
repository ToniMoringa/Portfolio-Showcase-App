# Aero Studio - Creative Agency Portfolio

A modern, responsive Single Page Application (SPA) built with React for showcasing creative agency projects. Features dynamic portfolio management, project search, case study modals, and an engaging studio story section.

## Features

- **Dynamic Portfolio Management** - Add new projects that persist using localStorage
- **Search Functionality** - Filter projects by title in real-time
- **Case Study Modals** - View detailed project information in a modal overlay
- **Studio Story Section** - Expandable "About Us" section with animated highlights
- **Frutiger Aero Styling** - Unique glassmorphism design with gradient effects
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Persistent Storage** - User-added projects survive page refresh

## Tech Stack

- React 18
- CSS3 (Glassmorphism, Gradients, Animations)
- Lucide React Icons
- LocalStorage API
